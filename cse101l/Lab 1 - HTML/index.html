Welcome


This week’s lab is to read through this HTML tutorial and complete the included exercises.

Setup


The script that makes your work visible on the internet requires you to save your files in a specific location.

  1. Double click My Computer on your desktop
  2. Open your Z: drive
  3. Create a folder named class — it must be all lowercase
  4. Open class
  5. Create a folder named cse101 — the letters must be all lowercase
  6. Open cse101
  7. Create a folder named lab1 — the letters must be all lowercase

Save all of your files into the lab1 folder that you just created.

Lab


Read the entire lab and complete all exercises. As mentioned above, save everything into the lab1 folder.

Put it online


The CSE server runs a web server (Apache) and is configured to allow each user to have a web site.

The first step in getting your work online is to create a public_html directory. I have written a script that will do the actual work for you, but you must connect to the Linux server and run the script yourself.

  1. Double click SSH Secure Shell Client on the desktop
  2. Click Quick Connect on the toolbar
  3. Enter cse.unl.edu as the server
  4. Enter your username and hit enter
  5. If prompted about a key, click Yes
  6. Enter your password when prompted
  7. Type the following command and hit enter; it must all be on one line and typed exactly as shown:

~rnelson/bin/cse101/makepublic.sh

Now you have a public_html folder in which to store your files. The files still need to be copied over and their permissions must be set. Again, I have written a script to do all of this for you. Run it like you did the makepublic script; the command is:

~rnelson/bin/cse101/lab1.sh
