For fall 2008, the CSCE 101 lab used PageTutor’s HTML tutorial for the first few labs to help introduce students various programming concepts, such as opening/closing symbols that match ( and , for example) and the fact that not everything typed in appears on the page.

While the tutorial worked reasonably well, it was somewhat lacking for a lab environment. Being designed to help a person learn to create their own pages, it isn’t set up with clear step-by-step instructions and information about when to start new files or erase the content in the existing one. To make the labs go smoother in the future, I found a CC-licensed tutorial to expand and make lab-friendly. All images used within the tutorial are also CC-licensed.

I admit, I’m cheating. This tutorial is done in RapidWeaver. It generates sane XHTML 1.0 Strict and makes the task of theming the tutorial much easier. The other simple alternative is to use PHP and some require_once() calls to do the theme, but this prohibits easy creation of a zip file that users can download to take the tutorial with them or easily modify it. Said zip and the RapidWeaver file are both available on the download page.
