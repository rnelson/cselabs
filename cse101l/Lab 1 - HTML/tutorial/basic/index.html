Basic

In the Introduction, we created a very simple web page and described the fundamental properties of HTML tags and attributes.

In this section, we'll start covering some of the basic HTML tags and attributes in a systematic way. Each subsection will display a short web page, which we will alter several times in order to show the effect of various tags. You should be comfortable with the idea of saving HTML to a file, altering it with a plain text editor, and viewing the results in your browser.

Section Goals

When you're done with this section, you should be able to:

  • Create a document with headings, paragraphs, and line breaks
  • Change the text color and background color
  • Use physical styles and logical styles to change the appearance of your text
  • Display images and create links to other web pages.

Subsections

Paragraph Breaks
In this section, we learn about the paragraph tag and the line break tag. We'll also discuss the reason why it's important to close your paragraph tags.
Headings
In this section, we cover the heading tags, and discover why it's really important to close them properly.
Font Styles
In this section, we take a look at tags that help you control the font. We'll touch on the difference between inline and block tags, and compare physical style tags versus logical style tags.
Colors
In this section, we learn how to change the background and foreground color of an entire page, a paragraph, or even a single word. We'll see examples of color names and hex codes, and we'll discuss which method is preferable.
Links
In this section, we break down a URL into its constituent parts, explain relative links vs. absolute links, provide examples of how to use named anchors, and more.
Images
In this section, we take a look at images: how to embed them in a web page, how to stretch and squash them, and how to "float" them with respect to the other content on the page.
Lists
In this section, we cover lists: ordered lists, unordered lists, and definition lists.
