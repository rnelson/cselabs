Lists

HTML provides several ways to display lists of information:

  • unordered lists (bulleted lists -- like this one)
  • ordered lists (numbered lists, procedures, formal outlines)
  • definition lists (a list of terms with definitions)

Lists are different from anything we've covered in the Introduction or Basic sections, because generating a list requires at least two tags working together.

Ordered Lists

Ordered lists start and end with a <ol> tag. Within the <ol> and </ol>, each list item is defined by an <li> tag, as follows:

Ordered List (source)

<h3>My Dot-com Business Plan</h3>
<ol>
  <li>Buy domain name</li>
  <li>Put up website</li>
  <li>???</li>
  <li>Profit!!</li>
</ol>

The result is an automatically numbered and indented list.

Ordered List (Results)

My Dot-com Business Plan

  1. Buy domain name
  2. Put up website
  3. ???
  4. Profit!!

The default ordered list uses arabic numerals (1, 2, 3, ...). However, you can alter this behavior fairly easily. Furthermore, between the <li> and </li> tags you may place nearly any HTML content that you want: text, paragraphs, images, even another HTML list.

Nested Ordered List (source)

<h3>Schedule for Monday</h3>
<ol style="list-style-type: upper-roman">
  <li>Suppress Gaul
    <ol style="list-style-type: upper-alpha">
      <li>Two more legions or three?</li>
      <li>Where to put victory arch?  Forum's 
        looking crowded...</li>
    </ol>
  </li>
  <li>Have Cicero killed</li>
  <li>Lunch</li>
  <li>Head-and-hands-on-pike viewing at
    the Forum (casual dress ok?)</li>
</ol>

We've specified a nested list. The outer list will use upper-case Roman numerals, and the inner list will start counting independently, using upper-case letters.

Nested Ordered List (Results)

Schedule for Monday

  1. Suppress Gaul
    1. Two more legions or three?
    2. Where to put victory arch? Forum's looking crowded...
  2. Have Cicero killed
  3. Lunch
  4. Head-and-hands-on-pike viewing at the Forum (casual dress ok?)

There are a large number of available list-style-types for ordered lists. The most common and well-supported values are upper-roman, lower-roman, upper-alpha, lower-alpha, and the default, decimal. Of course the official specification provides many more options, such as hebrew, georgian, and hiragana-iroha. Unfortunately, many browsers don't support the more obscure values. In fact, even the common values arent supported by the oldest browsers, but that's OK: the browser will just fall back to the default (decimal numbers).

Unordered Lists

Unordered lists are similar to ordered lists, except that they do not proceed in sequence. Unordered lists even recycle the <li> tag, which makes it very easy to turn an ordered list into an unordered list, and vice-versa. All you need do is turn your <ol> tags into <ul> tags:

Unordered List (source)

<h3>My Dot-com Business Plan</h3>
<ul style="list-style-type: square; color: #009900">
  <li>Buy domain name</li>
  <li>Put up website</li>
  <li style="list-style-type: disc; color: #ff0000">???</li>
  <li>Profit!!</li>
</ul>

In the parent tag (<ul>) we've specified that the list should be green and that the bullets should be squares. We've overridden this behavior in the third bullet point, specifying a red, circular bullet. Notice how the stylesheet behavior cascades from the parent tag to the child tags -- all the bullets take on the characteristics of the parent, unless we specify otherwise.

Unordered List (Results)

My Dot-com Business Plan

  • Buy domain name
  • Put up website
  • ???
  • Profit!!

Save links.html as list.html.

The lists can be ordered or unordered.
